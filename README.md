---

## 📌 Tópicos Abordados

Abaixo estão os módulos e conceitos explorados ao longo dos estudos:

| Módulo | Descrição | Status |
| :--- | :--- | :---: |
| 🔰 **1. Introdução ao SQL** | Compreensão do que é SQL e sua relevância no gerenciamento de bancos de dados relacionais (SGBDs). | 🚀 |
| ⚙️ **2. Instalação & Configuração** | Procedimentos para instalar, configurar o ambiente e conectar às ferramentas de SQL. | 🚀 |
| 📊 **3. Comandos Básicos (DML/DDL)** | Manipulação fundamental de dados utilizando comandos como `SELECT`, `INSERT`, `UPDATE` e `DELETE`. | ⏳ |
| 📐 **4. Estrutura de Tabelas** | Modelagem de dados, definição de colunas, linhas, e integridade com **Chaves Primárias (PK)** e **Estrangeiras (FK)**. | ⏳ |
| 🔍 **5. Consultas Avançadas** | Construção de queries complexas utilizando funções de agregação, subconsultas e junções (`JOINs`). | ⏳ |
| 💼 **6. Projetos Práticos** | Aplicação do conhecimento em cenários do mundo real, criando e gerenciando bancos de dados do zero. | ⏳ |
